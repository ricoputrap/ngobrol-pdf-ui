# Ngobrol PDF UI

Ngobrol PDF UI is a simple web-based chat application that allows users to have a conversation with a chatbot about the content of a PDF document.

## Features

1. Upload a single PDF file on a single chat session.
2. Ask about everything in the chat session about the content of the PDF document.
3. See the chat history of the current session.
4. See all chat sessions and re-enter the chat session.
5. Delete a chat session.
